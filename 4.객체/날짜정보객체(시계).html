-- getFullYear() 로컬 시간의 연도를 반환 2020 getYear() 로컬 시간의 연도를 반환 * (실제 연도 - 1900) 의 결과 반환 120 getMonth() 로컬 시간의 월 값을 반환 * 1월은 0, 2월은 1으로 각각의 달에 -1이 되어있음 getDate() 로컬 시간의 일 값을 반환 24 getDay() 로컬 시간의 주 기준 일 값을 반환 * 일요일은 0, 월요일은 1... 6 getTime() 로컬 시간의 전체 시간값을 밀리초 값으로 나타내어 반환 * 기준은 1970년 1월 1일 0시 0분 getHours() 로컬 시간의 시간 값을 반환 * 24시간제로 표시 21 getMinutes() 로컬 시간의 분 값을 반환 getSeconds() 로컬 시간의 초 값을 반환 getMilliseconds() 로컬 시간의 밀리초 값을 반환
